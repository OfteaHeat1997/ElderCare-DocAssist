# Frontend - React Native App

React Native application for ElderCare-DocAssist.

## Structure

- `app/` - Entry point and routes (Expo Router)
- `screens/` - Main screens: Record, Processing, Review, Approve
- `components/` - Reusable UI components
- `lib/` - Helper functions (storage, formatting)
- `assets/` - Icons, images, fonts
- `types/` - TypeScript type definitions (SOAP, vitals)

## Setup

1. Copy `.env.example` to `.env`
2. Install dependencies: `npm install`
3. Run: `npx expo start`

## Environment Variables

See `.env.example` for required configuration.
